Данный скрипт запускается один раз и за один раз приненяет следующие настройки:

1. Меняет имя машины
2. Вносит новое имя машины в /etc/hostname
3. Поднимает опущенные по умолчанию интерфейсы
4. Создаёт демона, который при каждом запуске поднимает интерфейсы

Теперь по файлам:
if.sh - скрипт, который будет добавлен в автозагрузку, чтобы по умолчанию поднимать все выключенные сетевые интерфейсы.
Помио этого будет скопирован в ранее созданную директорию/opt/scripts
up-interface.service - файл для systemd, который нужен для создания демона, который будет добавлен в автозагрузку.
settings.sh - Основной скрипт, который интерактивно выполнят настройки.

ВНИМАНИЕ!!!
Во время использования должен быть доступ к консоли, так как соединение SSH может быть разорвано после замены имени сервера.
