Данный скрипт запускается один раз и за один раз приненяет следующие настройки:

1. Меняет имя машины
2. Вносит новое имя машины в /etc/hostname
3. Поднимает опущенные по умолчанию интерфейсы
4. Создаёт демона, который при каждом запуске поднимает интерфейсы

Теперь по файлам:
if.sh - скрипт, который поднимет все интерфейсы и будет скопирован в /opt/scripts
up-interface.service - файл, который будет скормлен systemd и влключен как демон
settings.sh - скрипт, который нужно будет запустить.
